# Vanguard Gallery

The showcase site for **Project Vanguard** — valantic's initiative to ship production-grade AI tools for consulting workflows. The gallery presents each tool as a card in an interactive carousel, linking visitors directly to the source repository.

The carousel is fully data-driven: each tool's card is defined by a `vanguard-usecase.json` file that lives in the tool's own repo and is pushed here automatically via a GitHub Action whenever the file changes.

**Want your tool featured?** See the [onboarding guide](docs/ONBOARDING-USE-CASE.md) — setup takes about 10 minutes and only requires adding one secret and one workflow file to your repo.

**Have a news item to share?** See [ONBOARDING-NEWS.md](docs/ONBOARDING-NEWS.md) to publish a release announcement, field note, or talk to the news feed in minutes.
